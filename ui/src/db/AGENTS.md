# Database Layer

This directory contains the Drizzle ORM database client and schema.

## Key files
- `index.ts` — Drizzle client configured for Turso (libSQL)
- `schema.ts` — Auto-generated by Better Auth CLI, contains user/session/account/verification tables

## Commands
- `npm run db:push` — Push schema changes to Turso
- `npm run db:generate` — Generate migration files
- `npm run db:migrate` — Run migrations
- `npm run db:studio` — Open Drizzle Studio GUI

## Database
- Provider: Turso (SQLite cloud, libSQL protocol)
- ORM: Drizzle ORM with turso dialect
- Schema changes: Edit schema.ts, then run `npm run db:push`
